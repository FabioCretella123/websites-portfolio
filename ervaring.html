Project 01

Portfolio Website

Voor mijn portfolio heb ik een persoonlijke website gebouwd om mijn projecten en vaardigheden te tonen. Ik heb gebruik gemaakt van HTML en CSS om een responsive design te creëren dat er goed uitziet op desktop en in de toekomst ook op mobile. De website heeft een donker thema, een vaste sidebar-navigatie en een bovenste navigatiebalk met links naar externe platformen. Deze portfolio is nog niet volledig afgewerkt, als je enkele fouten opmerkt dan kan je me daar altijd voor contacteren via het contactformulier.
